    **Create `src/DynamicWeb.Serializer/Models/LogFileSummary.cs`:**
    ```csharp
    namespace DynamicWeb.Serializer.Models;

    public record LogFileSummary
    {
        public string Operation { get; init; } = "";
        public DateTime Timestamp { get; init; }
        public bool DryRun { get; init; }
        public List<PredicateSummary> Predicates { get; init; } = new();
        public int TotalCreated { get; init; }
        public int TotalUpdated { get; init; }
        public int TotalSkipped { get; init; }
        public int TotalFailed { get; init; }
        public List<string> Errors { get; init; } = new();
        public List<string> Advice { get; init; } = new();
    }

    public record PredicateSummary
    {
        public string Name { get; init; } = "";
        public string Table { get; init; } = "";
        public int Created { get; init; }
        public int Updated { get; init; }
        public int Skipped { get; init; }
        public int Failed { get; init; }
        public List<string> Errors { get; init; } = new();
    }
    ```

    **Create `src/DynamicWeb.Serializer/Infrastructure/LogFileWriter.cs`:**
    Static class with methods:
    - `CreateLogFile(string logDir, string operation)` -> returns full path using `$"{operation}_{DateTime.Now:yyyy-MM-dd_HHmmss}.log"`
    - `WriteSummaryHeader(string logFile, LogFileSummary summary)` -> writes `=== SERIALIZER SUMMARY ===\n{json}\n=== END SUMMARY ===\n` using `System.Text.Json.JsonSerializer` with `WriteIndented = true` and `PropertyNamingPolicy = JsonNamingPolicy.CamelCase`
    - `AppendLogLine(string logFile, string message)` -> appends `[{DateTime.Now:yyyy-MM-dd HH:mm:ss.fff}] {message}\n`
    - `ParseSummaryHeader(string logFile)` -> reads file, extracts text between the markers, deserializes to `LogFileSummary?`. Returns null if markers not found.
    - `GetLogFiles(string logDir)` -> returns `FileInfo[]` sorted by LastWriteTime descending (most recent first)

    **Create `src/DynamicWeb.Serializer/Infrastructure/AdviceGenerator.cs`:**
    Static class with `GenerateAdvice(OrchestratorResult result)` returning `List<string>`. Pattern matching on error strings per D-15:
    - Contains "FOREIGN KEY" (case-insensitive) -> FK ordering advice
    - Contains "group" AND "not found" -> missing group advice
    - Contains "duplicate" -> duplicate key advice
    - Any other error -> generic "Error in {TableName}: {error}"
    - If any result has Failed > 0 -> add re-run idempotency advice
    - Deduplicate with `.Distinct().ToList()`

    **Create test files** with tests covering all behaviors listed above. Use temp directories for LogFileWriter tests (clean up in Dispose). Use mock OrchestratorResult/ProviderDeserializeResult for AdviceGenerator tests.
