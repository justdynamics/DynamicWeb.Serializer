Add a private helper method `CreateColumnCheckboxList` to `PredicateEditScreen` that:

3. If `tableName` is null or whitespace, returns the editor with `Explanation = "Enter a table name to see available columns."`

7. Parses `currentValue` by splitting on `\r` and `\n` (same as SavePredicateCommand), trims, filters empties, and sets `editor.Value` to the resulting `List<string>` (per D-03: pre-check existing exclusions)
8. Wraps the `DataGroupMetadataReader` call in try/catch, setting `Explanation = "Could not query database columns."` on failure
