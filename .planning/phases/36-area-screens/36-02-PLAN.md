ExcludeAreaColumns applies ONLY to the `[Area]` SQL table columns in ReadAreaProperties. They must NOT be merged into a single set globally.

After the existing `excludeFields` HashSet construction (around line 73-74), add a new block to build the area column exclusion set:
```csharp
// Build area column exclude set (separate from item field excludes, per D-02)

    ? new HashSet<string>(predicate.ExcludeAreaColumns, StringComparer.OrdinalIgnoreCase)
    : null;
```

Update the `_mapper.MapArea()` call (around line 102) to pass the new parameter:
```csharp
var serializedArea = _mapper.MapArea(area, serializedPages, excludeFields,
    _configuration.ExcludeFieldsByItemType, excludeAreaColumns);
```
