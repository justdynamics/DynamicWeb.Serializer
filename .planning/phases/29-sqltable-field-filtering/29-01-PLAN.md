<!-- SqlTableWriter inherent skip guard: -->
From src/DynamicWeb.Serializer/Providers/SqlTable/SqlTableWriter.cs line 41:
```csharp
var itemColumns = allColumns
    .Where(col => row.ContainsKey(col))
    .ToList();
```
This means columns NOT in the row dict are excluded from the MERGE statement.
If excludeFields removes columns from YAML during serialize, those columns will be absent
from the row dict on deserialize, and BuildMergeCommand naturally skips them.
