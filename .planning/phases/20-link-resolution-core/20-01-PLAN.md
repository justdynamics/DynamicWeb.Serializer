    GREEN phase: Implement InternalLinkResolver with:

    1. Constructor: `InternalLinkResolver(Dictionary<int, int> sourceToTargetPageIds, Action<string>? log = null)`
       - Stores map and log callback. Initializes `_resolvedCount = 0`, `_unresolvedCount = 0`.

    2. Private static field:
       ```csharp
       private static readonly Regex InternalLinkPattern = new(
           @"(Default\.aspx\?ID=)(\d+)",
           RegexOptions.IgnoreCase | RegexOptions.Compiled);
       ```

    3. `public string? ResolveLinks(string? fieldValue)`:
       - If null or empty, return fieldValue as-is.
       - Use `InternalLinkPattern.Replace(fieldValue, MatchEvaluator)` where the evaluator:
         - Parses `match.Groups[2].Value` as int (sourceId)
         - If sourceId in map: increment `_resolvedCount`, return `match.Groups[1].Value + targetId`
         - Else: invoke `_log` with warning message including sourceId, increment `_unresolvedCount`, return `match.Value`

    4. `public static Dictionary<int, int> BuildSourceToTargetMap(List<SerializedPage> pages, Dictionary<Guid, int> pageGuidCache)`:
       - Create empty `Dictionary<int, int>`.
       - Call recursive helper `CollectSourcePageIds(pages, pageGuidCache, map)`.
       - Return map.

    5. Private static `CollectSourcePageIds(List<SerializedPage> pages, Dictionary<Guid, int> pageGuidCache, Dictionary<int, int> map)`:
       - For each page: if `page.SourcePageId.HasValue` and `pageGuidCache.TryGetValue(page.PageUniqueId, out var targetId)`, add `map[page.SourcePageId.Value] = targetId`.
       - Recurse into `page.Children`.

    6. `public (int resolved, int unresolved) GetStats() => (_resolvedCount, _unresolvedCount);`
