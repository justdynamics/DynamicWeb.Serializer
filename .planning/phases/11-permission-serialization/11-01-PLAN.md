    **3. Create PermissionMapper** (`src/.../Serialization/PermissionMapper.cs`):
    A helper class with these static/instance methods:
    - `static readonly HashSet<string> RoleIds = new(StringComparer.OrdinalIgnoreCase) { "Anonymous", "AuthenticatedBackend", "AuthenticatedFrontend", "Administrator" }`
    - `static bool IsRole(string ownerId)` -- checks RoleIds
    - `static string GetLevelName(PermissionLevel level)` -- switch: None=>"none", Read=>"read", Edit=>"edit", Create=>"create", Delete=>"delete", All=>"all", default=>level.ToString().ToLowerInvariant()
    - `List<SerializedPermission> MapPermissions(int pageId)` -- the main method:
      1. Call `new PermissionService().GetPermissionsByQuery(new PermissionQuery { Key = pageId.ToString(), Name = "Page" })` (per D-09)
      2. For each Permission returned:
         - If `IsRole(permission.OwnerId)`: Owner = permission.OwnerId, OwnerType = "role", OwnerId = null
         - Else (group): Look up group name via `Dynamicweb.Security.UserManagement.User.GetUserByID(int.Parse(permission.OwnerId))` -- if found and user.IsGroup, Owner = user.Name, OwnerId = permission.OwnerId, OwnerType = "group". If not found (deleted group per D-08), Owner = permission.OwnerId (raw numeric), OwnerId = permission.OwnerId, OwnerType = "group"
         - Level = GetLevelName(permission.Level), LevelValue = (int)permission.Level
      3. Return the list (empty list if no explicit permissions)
    - Add optional `Action<string>? _log` constructor parameter for logging

    **4. Add tests:**
    - In `PermissionSerializationTests.cs`: Test GetLevelName for all 6 values. Test IsRole for the 4 roles + a numeric string. Test that YAML output with permissions contains expected keys. Test that YAML output without permissions has no "permissions" key.
    - In `YamlRoundTripTests.cs`: Add test `Yaml_RoundTrips_PageWithPermissions` -- create page with 2 SerializedPermission entries, serialize, deserialize, assert all fields match.

    - In `DtoTests.cs`: Add test `SerializedPage_Permissions_DefaultsToEmptyList` and `SerializedPermission_CanBeConstructed`.
