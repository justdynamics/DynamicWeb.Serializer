<objective>
Add config-driven XML pretty-printing to the SqlTable serialization pipeline. SqlTable predicates gain an `xmlColumns` config list specifying which columns contain XML. On serialize, listed columns are pretty-printed via XmlFormatter.PrettyPrint. On deserialize, they are compacted via XmlFormatter.Compact. FlatFileStore gets ForceStringScalarEmitter so pretty-printed XML emits as YAML literal block scalars.

Purpose: Make XML-containing SQL table columns (DashboardWidget settings, EcomFeed config, EcomShippings XML, etc.) readable in YAML files for meaningful git diffs, completing XML-02.
Output: Updated predicate config model, FlatFileStore with literal block support, SqlTableProvider with XML format/compact in serialize/deserialize paths.
</objective>
